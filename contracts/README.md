# API contracts

`openapi.json` is generated by the backend build (Swashbuckle, see `docs/DECISIONS.md`) — never hand-edit it.

## Regenerating the panel's TypeScript client

```
cd backend && dotnet build Evo.sln   # refreshes contracts/openapi.json
cd panel && npm run generate-api-client
```

This writes `panel/src/api/generated/schema.ts` from `contracts/openapi.json` via `openapi-typescript`.

`panel/src/api/generated/` is **generated code**: gitignored, excluded from eslint, and must never be hand-edited.
CI (see `.github/workflows/ci.yml`) regenerates it and fails the build if the result differs from what's committed/expected — the drift check.
